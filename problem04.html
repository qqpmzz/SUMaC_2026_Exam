1. 크랭크 함수

빨간색 크랭크 R(n)
  • n < p 이면: n + 1
  • n = p 이면: 1
R(n) = (n mod p) + 1
검은색 크랭크 B(n)
  • n ≤ p - k 이면: n + k
  • n > p - k 이면: n - (p - k)
B(n) = ((n - 1 + k) mod p) + 1

2. 반복 연산

빨간 크랭크 k회:
Rk(n) = n + k (mod p)
검은 크랭크 k회:
Bk(n) = n + k2 (mod p)

3. Successful 조건

크랭크를 충분히 돌리면 모든 숫자(1~p)를 방문할 수 있다.

Successful ⟺ gcd(k, p) = 1

빨간색은 항상 Successful (주기 = p). 검은색은 gcd(k,p) = 1일 때만 Successful.

4. Special 조건

두 경로의 결과가 모든 시작 상태에서 동일하다:

  • 경로 1: Toggle → Crank (1회)
  • 경로 2: Crankk → Toggle (k회 크랭크 후 토글)
시작 색 경로 1 결과 경로 2 결과 조건
빨강 n + k n + k 항상 만족
검정 n + 1 n + k2 k2 ≡ 1 (mod p)
Special ⟺ k2 ≡ 1 (mod p) ⟺ k = 1 또는 k = p - 1
